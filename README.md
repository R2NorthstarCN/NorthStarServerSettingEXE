# NS ServerConfig Editor
可视化的北极星服务端服务器配置程序  BY：HK560

- 可配置一系列基础参数
- 为NSCN后续版本支持Unicode字符做转换支持，如转换中文字符等
- 后续加入支持dedi文件修改和更多可选项
  
## 使用方法
1. 给游戏安装北极星客户端，适用CN版和原版
2. 把本程序放到游戏根目录下
3. 运行本程序设置即可
